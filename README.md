# Async Race

Async race application that allows user to manage the collection of the cars, operate its engines, and show races statistics.  
This application allows to create, edit and delete cars in garage or generate cars.  
Launch single car and stop it.  
Launch and stop a race.  
See statistics.

![Application screenshot](https://github.com/CarphatianSnake/async-race/assets/17194679/75d0690f-1021-4458-bc8d-aa074c92e451)

## How to launch

To launch the application you need to clone/fork repo, install all packages and launch the [server mock](https://github.com/mikhama/async-race-api) on port `3000`.

## Stack

- TypeScript
- HTML
- SCSS
- Webpack
